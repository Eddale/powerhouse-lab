# Assets Folder

Put binary files, images, logos, and other non-text assets here.

**Why use assets?**
- Keep binary files separate from code
- Store logos, images, diagrams
- PDFs and other documents

**Example uses:**
- `logo.png` - Brand assets
- `diagram.svg` - Visual guides
- `template.xlsx` - Spreadsheet templates

Delete this README when you add your first asset file.
