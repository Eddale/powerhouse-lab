## QUICK ANSWERS

### "How long to build Phase 1?"
**30-60 minutes** if you follow the implementation guide

### "Do I need to code?"
**Phase 1:** No, just copy/paste templates
**Phase 2/3:** Yes, Python + Gmail API

### "What if I don't use Obsidian?"
Use any note-taking app with:
- File-based notes (Markdown)
- Template support
- Search/query capability

Examples: Notion, Logseq, Apple Notes (basic)

### "Can I skip to Phase 2?"
You CAN, but you SHOULDN'T.

Phase 1 teaches you the workflow. You need to know what to automate before you automate it.

### "What if the manual system works fine?"
**Then stick with it!**

Automation is NOT the goal. The goal is "never drop the ball."

If Phase 1 achieves that, you're done. Ship it and move on.

### If You Get Stuck

**Obsidian issues:**
- Check Dataview plugin is installed
- Verify folder paths match exactly
- Test with simple query first

**iOS Shortcut issues:**
- Start with minimal version (just 3 inputs)
- Test file creation in Files app first
- Check Obsidian Advanced URI plugin

**Workflow issues:**
- Are you logging too much? (Be selective)
- Are expected dates realistic? (3-5 days, not 1)
- Are you checking daily note? (Make it a habit)
